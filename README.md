# coders-of-delhi
A beginner-friendly Python project that demonstrates how social media platforms suggest: 👥 People You May Know 📄 Pages You Might Like using a JSON dataset and recommendation logic.


# Coders of Delhi - Data Analysis & Recommendation System

## Tasks

### Task 1: Load the User Data
- Read the JSON data file
- Validate the structure of the data
- Store the data for processing

### Task 2: Read and Display the Data Using Python
- Parse JSON data using Python
- Display user information in a readable format
- Handle missing or invalid entries

### Task 3: Identify Issues in the Data
- Detect duplicate records
- Identify missing values
- Find incorrect or inconsistent data entries
- Validate data formats

### Task 4: Clean the Data
- Remove duplicate records
- Fill or handle missing values
- Correct invalid data formats
- Standardize the dataset structure
